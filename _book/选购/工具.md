| 吸锡器                                                       | 1个  | 用于装配错误时解焊 |                                                              |
| 螺丝刀                                                       | 1个  | 适配M2十字螺丝     |                                                              |
| 斜口钳                                                       | 1个  | 用于剪灯脚         |                                                              |
| 镊子                                                         | 2个  | 用于测试主板       |                                                              |
